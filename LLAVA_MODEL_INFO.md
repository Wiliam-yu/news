# 🌋 LLaVA 模型介绍

**LLaVA: Large Language and Vision Assistant**

---

## 📊 模型概述

| 项目 | 信息 |
|------|------|
| **全称** | Large Language and Vision Assistant |
| **发布** | 2023 (NeurIPS Oral) |
| **团队** | Microsoft / UW Madison |
| **特点** | 开源多模态 GPT-4 级别能力 |

---

## 🎯 核心能力

### 1. 图像理解
- ✅ 物体识别
- ✅ 场景描述
- ✅ 文字识别 (OCR)
- ✅ 图表分析
- ✅ 截图理解

### 2. 视觉对话
- ✅ 问答对话
- ✅ 图像推理
- ✅ 多轮对话

### 3. 指令跟随
- ✅ 按要求分析图片
- ✅ 执行复杂视觉任务

## ⚡ 性能特点

### 优点
- ✅ 开源免费
- ✅ 本地运行，隐私安全
- ✅ 响应速度快 (相比云端)
- ✅ RTX 4060 可运行

### 局限
- ⚠️ 中文能力略弱于 Qwen
- ⚠️ 复杂推理不如 GPT-4
- ⚠️ 需要足够显存

## 🎮 在 Ollama 中使用

```powershell
# 拉取模型
ollama pull llava

# 运行
ollama run llava

# API 调用
curl http://localhost:11434/api/generate \
  -d '{"model": "llava", "prompt": "描述图片", "images": ["base64..."]}'
```
