## 🎮 使用场景

### 场景 1: 日常对话
```
Harry: 今天天气怎么样?
Tutor: Great question! Let's talk about the weather.
       It's a nice day today, isn't it?
       
       关键词: weather, nice day, sunny, cloudy
       尝试造句: "The weather is ___ today."
```

### 场景 2: 口语练习
```
Harry: [语音] (想表达"我想去餐厅")
Tutor: I heard you say something about going to a restaurant.
       Good start! 
       
       更自然的说法:
       - "I feel like going to a restaurant."
       - "I'm in the mood for dining out."
       
       一起练习: "I'm in the mood for ___."
```

### 场景 3: 写作批改
```
Harry: I very like this book.
Tutor: 语法小提示:
       ❌ I very like this book.
       ✅ I really like this book.
       ✅ I like this book very much.
       
       解释: "very" 通常不直接修饰动词,
       用 "really" 或 "very much" 代替。
```
