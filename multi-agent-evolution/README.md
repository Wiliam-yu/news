## 🎯 Agent 定义 (可扩展市场)

### 核心 Agent

| Agent | 角色 | 核心技能 |
|-------|------|----------|
| **CodeAgent** | 首席工程师 | github, skill-creator, xint |
| **SearchAgent** | 研究员 | brave-search, web-fetch, oracle |
| **CreateAgent** | 创意师 | voice-message, self-improving |
| **OpsAgent** | 运维官 | system-resource-monitor, openclaw-backup |
| **ReviewAgent** | 质量官 | clawsec, openclaw-mcp-debugger |

### Agent 市场 (Agent Market)
- 类似 LobeHub 的 Agent 市场
- 按需加载/卸载 Agent
- 支持自定义 Agent

## 🔄 协作模式

### 1. Agent 组 (Agent Teams)
```
Harry
  │
  ▼ 创建任务
Agent 组 (按需组建)
  │
  ├──▶ CodeAgent ──▶ 编写代码
  │         │
  │         ▼
  │    ReviewAgent ──▶ 审核 ──▶ 改进 ──▶ 完成
  │
  ├──▶ SearchAgent ──▶ 研究 ──▶ 报告
  │
  └──▶ CreateAgent ──▶ 创作 ──▶ 优化
```

### 2. 页面协作 (Pages)
- 多 Agent 在同一上下文并行工作
- 共享上下文和资源

### 3. 定时任务 (Schedule)
- Agent 定时执行任务
- 自动报告结果
