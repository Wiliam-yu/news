## 1️⃣ browser-use

### 基本信息

| 项目 | 值 |
|------|-----|
| ⭐ | 80,461 |
| 语言 | Python |
| 作者 | browser-use |

### 🎯 核心能力

| 能力 | 说明 | 边界 |
|------|------|------|
| **网页自动化** | 自动浏览、填写表单、点击按钮 | 需 Chromium |
| **数据抓取** | 提取网页内容、截图 | 动态内容有限制 |
| **任务执行** | "帮我投简历"类复杂任务 | 成功率依赖页面结构 |
| **多标签管理** | 同时控制多个浏览器标签 | 最多 5-10 个 |
| **云服务** | cloud.browser-use.com 提供托管 | 付费 |

### ⚠️ 限制与边界

| 限制 | 说明 |
|------|------|
| 依赖浏览器 | 需要安装 Chromium |
| 反爬虫 | 部分网站可能拦截 |
| 视觉理解 | 复杂交互可能失败 |
| API 费用 | 云服务需付费 |
| 速度 | 比直接 API 慢 |

### 🔧 安装要求

```bash
uv init && uv add browser-use && uv sync
# 需要 Python 3.11+
```
