# SKILL.md - Self-Improving Agent

## What
让 Agent 能够持续自我改进，通过反馈循环不断提升能力。

## Why
实现从"执行任务"到"优化执行方式"的跃迁。

## 核心机制

### 1. 反馈循环 (Feedback Loop)
```
执行 → 评估 → 改进 → 再执行 → ...
```

### 2. 指标追踪 (Metrics Tracking)
- 任务完成时间
- 尝试次数
- 用户满意度
- 错误类型

### 3. 模式识别 (Pattern Recognition)
- 识别重复任务模式
- 发现高频错误
- 优化工作流

### 4. 知识迭代 (Knowledge Iteration)
- 旧知识 → 新理解
- 失败 → 教训库
- 成功 → 最佳实践

---

## 实现方法

### 1. 任务日志
每次任务记录:
- 输入/输出
- 使用的方法
- 耗时
- 结果

### 2. 定期复盘
每小时/每天:
- 分析成功/失败模式
- 识别优化空间
- 更新执行策略

### 3. 技能更新
基于复盘结果:
- 优化提示词
- 创建新工具链
- 改进决策树

## 自我评估清单

每次任务完成后问自己:

1. ✅ 任务完成了吗?
2. ⚠️ 有没有更快的办法?
3. 💡 学到了什么新东西?
4. 🔄 下次能自动处理吗?
5. 📝 需要记录到记忆吗?

---

## 注意事项

- 进化是渐进的，不要追求一步到位
- 保留人类判断作为最终决策
- 定期回顾进化方向是否正确
